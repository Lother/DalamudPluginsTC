# DalamudPluginsTC

第三方 Dalamud 插件倉庫，彙整 [Lother](https://github.com/Lother) 為 TC（台服私人伺服器）維護的所有 Dalamud 插件 fork。

## 使用方式

在遊戲內開啟插件安裝器 → 實驗性 → 自訂插件庫，加入：

```
https://raw.githubusercontent.com/Lother/DalamudPluginsTC/main/repo.json
```

## 收錄插件

- EurekaHelper
- Accountant
- AutoRetainer
- Saucy
- LogogramHelper
- TriadBuddy (FFTriadBuddyDalamud)
- SomethingNeedDoing
- BossModReborn
- WrathCombo

所有插件皆為 TC 7.15 (Dalamud API Level 12) 相容分支，各自原始碼與版本紀錄請見對應的 `Lother/<PluginName>` repo。
